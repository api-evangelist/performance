BlazeMeter BlazeMeter is a self-service, load testing platform (PaaS) for developers providing an enterprise grade, out-of-the-box load testing solution. Start testing in under five minutes. Pingdom Pingdom is a service that tracks the uptime, downtime, and performance of websites. When problems happen with a site that Pingdom monitors, it immediately alerts the owner so the problem can be taken care of. Pingdom monitors websites from multiple locations globally so that it can distinguish genuine downtime from routing and access problems.
